# 优质 Workflow 项目

这里整理分享一些优质的 Agent Workflow 项目。

## 📌 关于 Agent Workflow

Agent Workflow 是指将多个 Agent 或 AI 组件按照特定的流程编排起来，形成自动化工作流的系统。常见的 Workflow 工具包括：

- **Dify** - 可视化 LLM 应用开发平台
- **n8n** - 工作流自动化工具
- **Flowise** - 拖拽式 LLM 应用构建器
- **LangFlow** - LangChain 可视化编排工具

## 🚀 项目列表

### [Dify工作流脚本合集](https://github.com/wwwzhouhui/dify-for-dsl)

**简介**：基于开源平台Dify的DSL工作流脚本合集，提供一键生成教学动画、票据识别、多语言文档翻译等现成的AI应用模板。

**技术栈**：Dify、DSL配置、LLM集成

**适合人群**：希望快速搭建AI应用的开发者、教育工作者、企业用户

**亮点**：
- 低代码、可视化开发体验
- 开箱即用的工作流模板
- 快速搭建定制化AI应用
- 涵盖教育、办公、数据处理等多个场景

### [Coze工作流200+合集](https://github.com/Hammer1/cozeworkflows)

**简介**：汇总超过200个即用型工作流，覆盖视频处理、社媒内容生成、数据分析等场景，如"全方位分析"、"文案仿写润色"等实用工作流。

**技术栈**：Coze平台、AI工作流编排

**适合人群**：内容创作者、新媒体运营、数据分析师

**亮点**：
- 200+现成工作流模板
- 场景丰富，拿来就用
- 专为内容生态打造
- 涵盖视频、图文、数据等多维度处理

---

### [n8n自动化模板500+](https://github.com/enescingoz/awesome-n8n-templates)

**简介**：500+个n8n自动化模板库，帮助打通Gmail、Slack、飞书等应用之间的数据壁垒，实现跨平台自动化工作流。

**技术栈**：n8n、API集成、Webhook

**适合人群**：需要提升工作效率的个人和团队、自动化爱好者

**亮点**：
- 500+跨平台自动化模板
- 连接主流办公和通讯工具
- 从邮件自动回复到定时推送新闻
- 显著提升个人和团队生产力
- 开源免费，可自托管部署

**简介**：收集超过500个AI Agent在各行业的应用案例，覆盖医疗、金融、教育、营销等领域，附带开源项目链接和完整实现思路。

**技术栈**：多种Agent框架、行业特定AI应用

**适合人群**：AI研究者、行业从业者、企业决策者、学生

**亮点**：
- 500+跨行业AI Agent案例
- 覆盖医疗、金融、教育、营销等多个领域
- 提供完整实现思路和开源代码
- 洞察AI如何颠覆传统行业
- 适合学习和实践参考
